Repaso páginas web

¿Qué es HTML y cuál es su función principal en el desarrollo web?

El lenguaje HTML sirve para describir la estructura básica de una página y organizar la forma en que se mostrará su contenido, además de que HTML permite incluir enlaces hacia otras páginas o documentos.

¿Qué son las etiquetas HTML y cómo se utilizan?

Son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo.

Proporciona ejemplos de algunas de las etiquetas HTML más comunes.

  • body: para el contenido.
  • head: para información sobre el documento.
  • div: división dentro del contenido.
  • a: para enlaces.
  • strong: para poner el texto en negrita.
  • br: para saltos de línea.
  • H1 … H6: para títulos dentro del contenido.
  • img: para añadir imágenes al documento.
