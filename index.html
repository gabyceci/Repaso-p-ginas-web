¿Qué es HTML y cuál es su función principal en el desarrollo web?

El lenguaje HTML sirve para describir la estructura básica de una página y organizar la forma en que se mostrará su contenido, además de que HTML permite incluir enlaces hacia otras páginas o documentos.

¿Qué son las etiquetas HTML y cómo se utilizan?

Son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo.

Proporciona ejemplos de algunas de las etiquetas HTML más comunes.

  • body: para el contenido.
  • head: para información sobre el documento.
  • div: división dentro del contenido.
  • a: para enlaces.
  • strong: para poner el texto en negrita.
  • br: para saltos de línea.
  • H1 … H6: para títulos dentro del contenido.
  • img: para añadir imágenes al documento.


    • ¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

    CSS se utiliza para dar estilo al contenido de una página web añadiendo elementos de diseño como colores, fuentes y espaciado.

    ¿Cuáles son las propiedades más importantes de CSS?

  • Display: Establece la visualización del elemento HTML, entre ellos están: inline, block, inline-block, flex y grid.
  • Font: Se refiere al tipo de letra, tamaño, estilo, etc.
  • Color del texto: Como lo indica estable el color del texto.
  • Margin: Asigna el margen para cada lado del elemento.
  • Padding:El padding de un elemento es la cantidad de espacio entre el borde y el contenido del elemento
  • Border: El estilo del borde del elemento.
  • Background: Color del elemento.
  • Width: Ancho.
  • Height: Alto.

    • Explica el modelo de caja (box model) en CSS.

    El modelo de caja CSS es un contenedor que contiene múltiples propiedades, incluidos los bordes, los márgenes, el relleno y el contenido en sí . Se utiliza para crear el diseño y la disposición de las páginas web. Según el modelo de caja CSS, el navegador web proporciona cada elemento como un prisma cuadrado.


    ¿Qué es el diseño responsivo y cómo se logra con CSS?

    Las imágenes responsivas son imágenes que se ajustan automáticamente al tamaño de la pantalla del dispositivo. Esto se logra mediante el uso de atributos HTML y CSS específicos que indican qué imagen se debe cargar según el tamaño de la pantalla.

    ¿Qué es JavaScript y cómo se integra en el desarrollo web?

    Es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.

    Explica cómo JavaScript interactúa con el HTML y CSS a través del DOM (Document Object Model).

    Permite a los desarrolladores manipular y gestionar la estructura, el estilo y el contenido de una página web de forma dinámica.


    Proporciona ejemplos de cómo JavaScript puede hacer que una página web sea interactiva.

    Utilizando eventos para ejecutar comandos de JavaScript en función de las acciones que se realicen en la pantalla . Un evento es una interacción: onclick, onload, onmouseover, onmouseout, onsubmit, onchange. A la derecha se incluye un script simple para ilustrar el uso de un evento.

    ¿Qué es Git y por qué es importante para el desarrollo web?

    Git proporciona herramientas para aislar los cambios y volver a combinarlos posteriormente y tener todo el proyecto en la nube pero todo esto por consola.

    Explica cómo Git se usa para el control de versiones en proyectos de desarrollo web.

    Permite hacer un seguimiento de los cambios en el código, lo que facilita la resolución de problemas y la coordinación del equipo, además es muy necesario ya que si sucede un error en alguna rama no afecta la rama principal.


    ¿Qué es GitHub y cómo facilita la colaboración en proyectos web?

    Es una plataforma en la nube que permite a los desarrolladores colaborar en proyectos de software. Utiliza el sistema de control de versiones Git para almacenar, compartir y administrar el código.
